# Mobile App Flow (Android/iOS)

1.  Install App
2.  Register via Teacher
3.  Login
4.  QR Scan
5.  Select ENTER
6.  Geo tracking ON
7.  Alerts & notifications
8.  Select EXIT
9.  Geo tracking OFF (except parent view)
10. Reports & history

Parents Flow: Login → View Child → Live Location → Reports → Alerts
