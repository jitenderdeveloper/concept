# Core Features

## Attendance

-   QR Based ENTER / EXIT
-   One ENTER + EXIT per day
-   Manual attendance (Teacher/Principal)
-   Late / Early reason system

## Geo Tracking

-   Live location after ENTER
-   Geo radius violation alerts
-   Exit & re-entry timeline

## Notifications

-   Admin → Teacher / Student
-   Teacher → Student
-   System alerts (Late / Geo / No Entry)

## Reports

-   Monthly attendance auto email
-   Performance & discipline analytics

## Logs

-   Login / Logout
-   Attendance
-   Geo movement
-   Device change
